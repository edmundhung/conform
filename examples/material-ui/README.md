# Material UI Example

This example shows you how to integrate [material-ui](https://mui.com/material-ui/) Inputs components with Conform.

| Component             | Native support | Integration required |
| --------------------- | :------------: | :------------------: |
| Text Field (input)    |      Yes       |                      |
| Text Field (textarea) |      Yes       |                      |
| Select (native)       |      Yes       |                      |
| Select (default)      |                |         Yes          |
| Autocomplete          |      Yes       |                      |
| Checkbox              |      Yes       |                      |
| Radio Group           |      Yes       |                      |
| Rating                |                |         Yes          |
| Slider                |                |         Yes          |
| Switch                |                |         Yes          |
