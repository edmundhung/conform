# Chakra UI Integration

This example shows you how to integrate [chakra-ui](https://chakra-ui.com/docs/components) forms components with Conform.

| Component   | Native support | Integration required |
| ----------- | :------------: | :------------------: |
| Input       |      Yes       |                      |
| Select      |      Yes       |                      |
| Textarea    |      Yes       |                      |
| NumberInput |                |         Yes          |
| PinInput    |                |         Yes          |
| Editable    |                |         Yes          |
| Checkbox    |      Yes       |                      |
| Switch      |      Yes       |                      |
| Slider      |                |         Yes          |
| Radio       |      Yes       |                      |
