# Chakra UI Example

This example shows you how to integrate [chakra-ui](https://chakra-ui.com/docs/components) forms components with Conform.

## Compatibility

> Based on @chakra-ui/react@2.4.2

**Native support**

- Input
- Select
- Textarea
- Checkbox
- Switch
- Radio
- Editable

**Integration required**

- NumberInput
- PinInput
- Slider
